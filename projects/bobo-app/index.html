bobo Web Application

branding / web

Skills: HTML, CSS/SCSS, JavaScript/jQuery, AJAX, HTML EMails. MySQL, PHP

Website: http://bobo-ny.com

Github: bobo

bobo New York was a app that was to launch in New York City in attempt to disrupt the growing problem in fine dining in the big city.

bobo was designed to have a minimal user interface that allowed for users to select reservations as quickly as possible. This ideal lead me to make it as a single page application that was desktop and tablet friendly. This is a custom PHP frame work that uses many AJAX calls to retrieve and store information in a MySQL Database.
